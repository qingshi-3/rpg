# CoreLoop

## Combat Loop

1. Enemies generate Intent, either public or semi-public.
2. Player turn:
   - Allocate AP.
   - Control heroes through movement and skills.
   - Adjust rules through Command.
   - Use cards optionally.
3. Execution phase:
   - Troops execute according to their rules.
   - Enemies execute Intent.
4. Resolution:
   - Resolve damage, status, and terrain.
5. Repeat.

## Core Decision Axes

- Space: position and movement.
- Resource: AP.
- Rules: command decisions.
- Enemy behavior: Intent.
- Randomness: cards.

## Combat Pace

- Normal combat: 3 to 5 turns.
- Elite combat: 6 to 8 turns.
