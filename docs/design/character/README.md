# Character Design Index

This directory stores character definition, emotion, and relationship design.

## Routes

- Character definition: `character-definition.md`
- Emotion and relationship system: `emotion-system.md`
- Emotion gameplay query contracts: `emotion-gameplay-query-contracts.md`

## Rule

Character and emotion systems should provide structured state, events, and query results. They should not directly operate battle flow or strategic world internals.
