# UnitSystem

## Hero Structure

Hero is composed from:

- Attributes.
- Ability set.
- Command capability.
- Passives.

## Hero Role Examples

- Warrior: control and frontline durability.
- Ranger: marking and focus fire.
- Mage: area control.
- Priest: healing and protection.
- Rogue: mobility and disruption.

## Race

Race is a tag system, not a cosmetic skin.

Supported Race tags:

- Human.
- Beast.
- Undead.
- Construct.
- Spirit.

Race is used for:

- Ability checks.
- Status immunity.
- Counter relationships.

## Faction

Faction is used for:

- Enemy composition.
- Behavior style.

## DamageType

Supported DamageType values:

- Physical.
- Fire.
- Shock.
- Holy.
- Shadow.

DamageType uses local counter rules. It should not use cyclic counters.

## Status Categories

- Control: Stun.
- Behavior: Mark, Guard.
- Duration: Burn, Poison.
- Command: Focused, Retreat.
