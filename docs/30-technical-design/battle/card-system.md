# CardSystem

## Role

Cards are extension tools for the command system.

Cards are tactical modifiers, not the main system and not the main damage source.

## Rules

- Cards consume AP.
- Cards have no independent resource.
- Cards must not dominate damage output.

## Card Types

### Tactical Cards

- Focus fire.
- Advance.
- Retreat.

### Buff Cards

- Inspire.
- Defensive formation.

### Intervention Cards

- Interrupt Intent.
- Forced movement.

### Control Cards

- Redraw.
- Select.

## Design Principles

- Change behavior.
- Change the battlefield situation.
- Do not stack raw numerical bonuses as the main value.
