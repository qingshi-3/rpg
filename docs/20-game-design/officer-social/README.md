# Officer Social Gameplay Index

This directory stores gameplay design for officer/person relationship management.

## Gameplay Role

This pillar provides the 三国志 / 三国立志传-style people layer: recruitment, appointments, loyalty, affinity, duty, rank, bonds, conflict, betrayal risk, promotion, and relationship-driven events.

## Routes

- Summoned character and abstract relationship model: `summoned-social-relationship-model.md`
- Unit talent, rank, control authority, and promotion: `unit-talent-rank-control.md`

## Expression Boundary

Summoning is a way to express recruitment and character arrival. It can explain why historical, legendary, and original characters appear together, but gameplay should still be designed as people acquisition, relationship management, appointment, loyalty, and authority.

## Technical Boundary

Emotion state, query contracts, and backend implementation belong in `../../30-technical-design/relationship/`.
