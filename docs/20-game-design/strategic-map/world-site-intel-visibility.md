# WorldSite 情报可见性

## 目标

战略层信息默认清晰，战术布阵信息按场域状态和遮蔽来源分层。场域探索只在能改变战斗或经营时出现，不成为每场战斗前的固定负担。

## 三种场域

- 透明场域：可直接查看场域地图、入口、资源点和基础布阵。
- 半透明场域：战略信息公开，内侧布阵、伏兵、侧路或机关部分未知。
- 遮蔽场域：有明确遮蔽来源，玩家需要潜入、侦察或承担盲打风险。

## 规则

- 低级场域可以透明，但必须在内容中显式配置为透明。
- 战略派遣所需信息不能隐藏。
- 隐藏必须说明原因。
- 探索结果必须改变战斗条件或经营资产。
- 已探索信息进入场域记忆。
- 半透明和遮蔽只影响战术细节，不阻断基础战略决策。
- 隐藏入口不能作为玩家可用战斗入口，直到探索记忆或完整战术可见性揭示它。
- 普通直接进入敌方场域只适用于完整战术情报可见的场域；需要探索的敌方场域必须通过抵达部队潜入。
- 探索点行动会写入场域记忆，并影响后续战斗请求的入口、战术标签和探索优势。
