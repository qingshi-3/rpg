# Production Index

This directory stores production state: roadmap, priorities, technical change notes, and open questions.

## Routes

- Roadmap and current priorities: `roadmap/README.md`
- Technical change notes: `technical-changes/README.md`
- Historical auto battle migration: `technical-changes/2026-05-16-auto-tactics-migration.md`
- Open questions: `open-questions/README.md`

## Production Layer Rule

Production documents track time-bound plans and accepted changes. They should not become the primary source of product positioning, gameplay rules, or technical contracts. When a decision becomes stable, consolidate it into the correct product, gameplay, technical, content, or QA document.
