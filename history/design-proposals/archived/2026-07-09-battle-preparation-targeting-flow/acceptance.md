## User Direction

The user rejected the current bottom battle-preparation plan bar as functionally unnecessary and approved replacing it with a single lower-right start-battle button. The user also approved making the post-placement destination step a focused map interaction: after placing a battle group, all normal UI remains hidden, a curved card-style arrow points from the placed group to the pointer, the system cursor is hidden, grid hover remains visible, and left-click confirms the destination.

The user explicitly chose left-click for destination confirmation during this state because the player is focused only on choosing a destination and should not need to switch mouse-button semantics.
