# Tile Texture Imports

Tile texture files in this directory are copied from the external asset library.

External source root:

```text
C:\Users\qs\asset\PNG\Tilesets
```

The external asset library is read-only. Do not modify, rename, move, or delete files in that source library.

Project imports should copy from each source package's `Tiled_files` directory into the matching package directory here.

Files inside this project may be reorganized as needed, but source assets outside the project must remain untouched.
