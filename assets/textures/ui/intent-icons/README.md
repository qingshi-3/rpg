# Intent Icon Assets

Intent marker icons live in this folder and are referenced by semantic keys through `BattleIntentIcons`.

Current sources:

- `attack.png`, `defense.png`, `support.png`, and `summon.png` use copied CraftPix assets from `C:\Users\qs\asset\rpg\Icons`.
- `advance.png`, `control.png`, `mobility.png`, `charge.png`, `retreat.png`, and `unknown.png` are cropped from generated concept sheets selected during UI review.

The external asset library is read-only. Edit, rename, or replace only copied files inside this project directory.

## Main Intent Keys

- `attack`: near-term threat / direct attack.
- `advance`: future threat / advancing.
- `defense`: defensive posture.
- `support`: support, healing, or protection.
- `control`: control or restriction.
- `mobility`: repositioning or special movement.
- `summon`: summoning units or objects.
- `charge`: preparing a stronger future action.
- `retreat`: retreating or escaping pressure.
- `unknown`: unreadable or hidden intent.
