# Professions, Tags, And Bonds Detail

## Parent Authority

Global rules live in `../../content-systems-long-term-design.md`, especially the professions, bonds, corps definition, and aptitude sections.

## Boundary

This detail area defines the content taxonomy that lets historical, fictional, and fantasy content share one ruleset:

- combat classes;
- fantasy forms;
- tags;
- profession and tag bonds;
- class behavior expectations;
- fantasy tag interaction rules.

## To Refine

- First-phase combat classes.
- First tag list and tag meanings.
- Which bonds affect tactical behavior versus numeric modifiers.
- How fantasy forms such as dragon, crab, undead, construct, plant, or machine map to classes and resources.

## Non-Goals

- Relationship-specific historical bonds as the main build system.
- Pure "deploy N of a tag for +Y%" as the main bond identity.
- Content origin directly owning combat rules.
