# Acceptance

Status: Accepted

## User Acceptance

On 2026-06-07, the user accepted the interactive battle-preparation prototype direction with the explicit confirmation: `符合预期`.

## Accepted Interaction Requirements

- The player drags the corresponding hero icon from a compact unit list.
- During drag, the battlefield shows the hero-led formation for that company.
- Legal placement renders normally.
- Illegal placement renders the whole formation red.
- After release, committed placement remains on the battlefield.
- The player views a battlefield thumbnail to select the current company's attack objective.
- The player selects the current company's attack/engagement strategy.
- During drag, persistent UI should slide offscreen and return after release so it does not block the map.
