# Acceptance

Accepted: 2026-06-07
Accepted By: User conversation

The user approved the two-layer formation model:

- strategic management can edit a company's long-term default formation;
- battle preparation uses a current-battle selected formation initialized from that default;
- drag deployment uses the selected formation;
- formation changes after placement must not corrupt the existing placement if the new shape cannot fit.

The user also reported that drag preview can overlap two `2x1` units into a three-cell footprint. The accepted design requires formation validation to prevent member footprint overlap.
