# Thunder Mark Skill Family Acceptance

Accepted: 2026-06-11

The user approved moving from discussion into implementation after converging on a four-skill thunder-mark family:

- projectile mark throw;
- teleport to a selected legal cell around a mark;
- channeled melee damage that can continue through teleport;
- later spatial transfer for units or skill events.

Implementation scope starts with the first three skills. Spatial transfer is accepted as a high-tier direction but remains deferred.
